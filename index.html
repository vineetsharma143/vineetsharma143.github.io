Projects

Projects

Below are the projects of Electrical Engineering on Model order reduction, Controller designing, Optimization techniques and IoT.

Frequency weighted and Frequency interval Gramian framework-based Model Reduction using Singular value decomposition

This work provides a novel structure of fictitious input and output matrices, resulting in the proposed continuous-time system Gramians. The suggested strategy provides stable ROMs even when input and output weightings are incorporated into the system. Furthermore, the proposed approach is extended with the frequency-interval Gramians, ensuring stable ROMs.

Confined Frequency-Interval Gramian Framework-Based Balanced Model Reduction

This work presents an innovative structure of confined frequency-intervals Gramians for continuous-time systems. The developed ROM ensures stability and minimal inaccuracy in approximation error over the desired frequency intervals. Also, the developed technique provides an easy-to-calculate error-bound.

An Innovative Frequency-Limited Interval Gramians-Based Model Order Reduction Method Using Singular Value Decomposition

This work proposes a new structure for stable continuous-time systems based on limited-interval Gramians. SVD-based Frequency Weighted Model Order Reduction of Continuous-time Systems

This work presents a new frequency-weighted model reduction method using an innovative structure of frequency-weighted Gramians. The suggested approach ensures ROM stability and minimal inaccuracy in frequency-response approximations. The numerical examples support the effectiveness of the proposed technique and compared to conventional methods, the suggested approach offers consistent results, illustrating its usefulness.

New input-output matrices-based frequency interval balanced truncation

The existing frequency-limited interval-based model reduction techniques can generate unstable models for a stable continuous-time linear system. Further, some current methods resolve the instability problem, yielding a reduced model with a significant approximation error. The suggested approach is based on novel limited-interval Gramians, ensuring stability and producing a low approximation error.
